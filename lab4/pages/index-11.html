Висновки

Під час виконання цієї практичної роботи я навчився працювати з масивами, взаємодії з користувачем шляхом введення через метод prompt та виведення через console.log та alert.

Окрім цього навчився форматувати рядки за допомогою змінних а також перевіряти чи відповідає рядок потрібному паттерну.

Наприкінці реалізував невеликий додаток, що дозволяє додавати підказки до текстового абзацу.

Під час роботи я також освоїв:

  1. Роботу з регулярними виразами для валідації часу та логінів
  2. Використання конструкції switch для обробки числових значень у текстовому вигляді
  3. Роботу з Map для зберігання та перевірки облікових даних
  4. Створення функцій з параметрами та поверненням значень
  5. Роботу з глобальними змінними для відстеження стану (totalSpent)
  6. Алгоритм сортування вибором для впорядкування масивів
  7. Роботу з двовимірними масивами (матрицями) та їх генерацією
  8. Обробку подій кліку та динамічне створення DOM-елементів

Особливо цікавим було реалізувати інтерактивний функціонал для додавання підказок до тексту, що демонструє практичне застосування JavaScript у веб-розробці.
